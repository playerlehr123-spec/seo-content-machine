# OBS Capture Guide (MixPulse Beta)

1. Open MixPulse in standalone or DAW.
2. Open the **Creator Output Window** (Output button).
3. In OBS, add **Window Capture**.
4. Select the MixPulse Creator Output window.
5. Resize/crop for 16:9, 9:16, 1:1, or 4:5 compositions.
6. Record in OBS for beta promo clips.
7. Use **Export Frame** (PNG) for thumbnails/stills.

## Limitations (beta)
- No transparent output yet.
- No NDI/Spout/Syphon yet.
- No built-in video recorder yet.
- Use OBS as the primary video capture path in beta.


Workflow polish:
- Open WaveFrame Output.
- Choose scene and output aspect.
- Add Window Capture in OBS and select WaveFrame Output.
- Record in OBS; use Frame for stills/covers.


## WaveFrame Export + OBS pairing
- Use Output aspect guides to frame capture scenes.
- Use Export Frame for still artwork/covers.
- Use OBS to record motion video.
- Recommended pairings: Reel Visualizer + 9:16, Stream Overlay + 16:9, Release Promo Card + 4:5, Label Drop + 1:1.

Limitations: no built-in video export yet, no transparent background export yet, no NDI/Spout/Syphon.
