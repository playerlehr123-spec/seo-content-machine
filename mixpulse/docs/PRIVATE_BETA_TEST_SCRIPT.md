# Private Beta Test Script

## Standalone test
1. Download `MixPulse-Windows-Beta`.
2. Extract ZIP.
3. Run EXE.
4. Open Settings.
5. Select input/output if needed.
6. Confirm meters move.
7. Click Output.
8. Click Full, then Esc.
9. Click HUD on/off.
10. Click Frame.
11. Confirm Frame does not crash and note whether a still PNG is written.
12. Leave app open 5 minutes.
13. Close/reopen.

## VST3 test
1. Copy `MixPulse.vst3` to VST3 folder.
2. Rescan DAW.
3. Load on audio track.
4. Confirm pass-through.
5. Confirm meters move.
6. Test Output, Full, HUD, Frame.
7. Add second instance.
8. Remove plugin.
9. Save/reopen project if safe.

## Result summary
- Pass / Fail
- Notes
- Artifact/run number
- Known issues found


## State recall checks
- Change scene/module/FX/HUD/export preset, save DAW project, reopen and verify recall.
- Save/load user preset and verify brand fields and logo path status.
