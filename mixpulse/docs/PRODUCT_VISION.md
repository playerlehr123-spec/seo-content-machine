# WaveFrame Product Vision

## Thesis
WaveFrame is a VST + standalone creative tool that turns audio into polished, branded social content.

## Positioning
Audio-reactive content tools for artists, DJs, labels, and streamers.

## Target users
- music producers
- DJs
- artists
- record labels
- streamers
- promo teams
- content creators
- visual artists

## Product pillars
1. Meter
2. React
3. Brand
4. Export
5. Templates
6. Community / Club

## Core workflow
Drop WaveFrame on audio → pick template → audio drives branded visual → export stills and capture motion for social channels.

## Reference lanes (conceptual only)
- compact metering utility lane
- audio-reactive visual engine lane
- design tool / template club lane

## Current scope
- analysis-only audio pass-through
- peak/rms/spectrum/HUD foundation
- visual rack + output window foundation
- frame PNG export + social aspect presets (beta)
- creator template/brand-layer scaffolding

## Future scope
- expanded tool suite modules
- richer creator templates/packs
- advanced export workflows
- community loops (inspiration/requests)

## What WaveFrame is not
- not a full video editor
- not a DAW timeline
- not a mastering analyzer
- not a cloud platform
- not a public release yet
