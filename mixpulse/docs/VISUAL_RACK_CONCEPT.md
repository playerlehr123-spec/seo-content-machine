# Visual Rack Concept (WaveFrame Output)

WaveFrame Output includes intentional idle visuals (dark neon, subtle grid, center glow) and responsive active visuals with lightweight FX toggles:
Bloom, Mirror, Colorize, Grid, Scanlines, ChromaticSplit, Vignette.

Placeholder render modes are presentation-ready stubs: Logo Reactor, Release Promo Card, Meter Overlay, Waveform, Vectorscope, Gradient Field, Particle Burst, Tunnel, Plasma.
