# MixPulse Visual Rack Concept

MixPulse is evolving into a mini meter plus audio-reactive creator visual rack.

## Vision
- Fast visual modules
- Lightweight FX chain toggles
- Simple modulation routing
- Creator output-first workflow

## Implemented now
- VisualRack state model with module/FX/mod/output enums
- Built-in preset scaffolding (in-memory)
- Renderer hooks so Spectrum Bars reacts to rack FX flags
- Placeholder-safe rendering for non-spectrum modules

## Placeholder / TODO
- Preset persistence (JSON)
- Advanced module UIs
- Export presets
- Spout/Syphon/NDI investigation (future)

## Not being copied
MixPulse is inspired by pro video engines in general, but is not copying any specific product, branding, UI, or code.

## Current scope limits
- No video timeline editor
- No video clip import
- No Spout/Syphon/NDI routing yet
- No MP4/WebM export yet
