# WaveFrame Beta Bug Reporting

Use this guide when filing private beta bugs.

## Required Fields
- QA level: Level 0, Level 1, Level 2, Level 3, Level 4, or Level 5.
- Severity: P0, P1, P2, or P3.
- Artifact/run number.
- Standalone, VST3, or both.
- DAW name/version if involved.
- Windows version.
- Audio interface if relevant.
- Module/template selected.
- Output window open: yes/no.
- OBS involved: yes/no.
- Repro rate: always, often, sometimes, once, not reproduced.
- Expected behavior.
- Actual behavior.
- Steps to reproduce.
- Whether audio pass-through changed.

## Crash Reports
Include what was clicked immediately before the crash and whether it reproduces after restart.

## Audio Reports
Audio reports are highest priority. State whether audio became louder, quieter, muted, clipped, delayed, distorted, filtered, compressed, widened, or otherwise changed.

## Visual / Output Reports
Include selected module/template, output preset/aspect if known, Output window status, OBS status, and screenshot/video when possible.
