# WaveFrame Beta Bug Reporting

Use this guide for v0.1 private beta candidate bugs.

## Include
- Artifact/run number.
- Windows version.
- DAW/version if applicable.
- Standalone or VST3.
- Module/template selected.
- Output window open: yes/no.
- OBS involved: yes/no.
- Steps to reproduce.
- Expected behavior.
- Actual behavior.
- Crash/freeze status.
- Whether audio pass-through changed.
- Screenshot, log, or short video if available.

## Severity
- P0: launch/build/artifact/audio pass-through/crash-on-open blocker.
- P1: Output, Frame, HUD, preset, module switching, or DAW remove/reload crash.
- P2: UI, visual, status, or docs issue.
- P3: polish wording/spacing/tooltip issue.

Audio pass-through changes are always high priority.
