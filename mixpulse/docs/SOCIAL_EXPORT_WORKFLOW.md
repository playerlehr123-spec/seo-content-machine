# WaveFrame Social Export Workflow

## Output now / near term
- PNG still export
- OBS capture for motion

## Output later
- video export
- SVG export for vector tools (if implemented)

## Presets and dimensions
- Current Window capture
- 9:16 (Reels/TikTok/Shorts)
- 1:1 square post
- 4:5 feed post
- 16:9 YouTube/stream
- album-art square (future)

## Use cases
- release announcements
- teaser clips (via OBS)
- thumbnails/covers
- label promo posts
- stream overlays

## Current limitations
- no built-in video export yet
- no MP4/WebM output yet
- no transparent export pipeline yet
